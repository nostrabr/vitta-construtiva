# Rotas Amigaveis - Site Publico

Este projeto usa rotas amigaveis apenas no site publico.
O admin nao usa essas rotas.

## Rotas publicas

- /
- /inicio
- /sobre-nos
- /contato
- /depoimentos
- /areas/{id}
- /projetos/{areaId}/{projetoId}

## Nginx (bloco 8080)

Adicionar estas regras no server da porta 8080:

```nginx
# Rotas amigaveis (somente site publico)
rewrite ^/inicio/?$ /index.php last;
rewrite ^/sobre-nos/?$ /sobre-nos.php last;
rewrite ^/contato/?$ /contato.php last;
rewrite ^/depoimentos/?$ /depoimentos.php last;
rewrite ^/areas/([0-9]+)/?$ /area-atuacao.php?id=$1 last;
rewrite ^/projetos/([0-9]+)/([0-9]+)/?$ /projeto.php?area=$1&id=$2 last;

try_files $uri $uri/ /index.php?$args;
```

## Checklist rapido

1. Testar configuracao: `nginx -t`
2. Recarregar servico: `systemctl reload nginx`
3. Testar no navegador as URLs acima
